See addition, subtraction, multiplication, factorial, combinations, and powers as pebbles.

Addition = putting the two piles together.
Subtraction = start with a pile of a and remove b pebbles.
Multiplication a √ó b = b rows of a pebbles (an area).
Factorial n! = number of all possible orders of n distinct pebbles.
Combinations = choose k pebbles out of n, order does not matter.
a^b = number of ways to choose 1 pebble at each of b positions, with a choices each time.
